# 🌌 Interplanetary Democratic Constitution (IDC) 
**Executable Civilization System & AGI Safety Framework**

### 📜 О проекте
IDC — это программный прототип "исполняемой конституции" для будущих внеземных поселений. Система переводит этические и правовые нормы с человеческого языка на язык программных инвариантов.

### 🧠 Ключевые модули
1. **Constitutional Layer:** Права на жизнь и свободу как `lambda`-инварианты.
2. **Liquid Democracy:** Система динамического делегирования голосов.
3. **AGI Tripwires:** Аппаратные и программные "растяжки" для контроля ИИ.
4. **Adaptive Economy:** Система саморегуляции ресурсов колонии.
